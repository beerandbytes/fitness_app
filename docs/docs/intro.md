Bienvenido a la documentación oficial de **Fitness App**.

Este proyecto es una aplicación completa de seguimiento de salud y entrenamiento que incluye:

- Un **backend Node.js/Express** con **Drizzle ORM** y **PostgreSQL**.
- Un **frontend React (Vite)** con una interfaz moderna y accesible.
- Una **base de datos relacional** con tablas para usuarios, ejercicios, nutrición, rutinas, notificaciones, logros, chat y más.

En esta documentación encontrarás:

- Cómo instalar y ejecutar el proyecto (localmente y en producción).
- Detalles de la arquitectura del backend y frontend.
- La estructura completa de la base de datos y todas sus relaciones.
- Referencia detallada de la API (endpoints, parámetros y respuestas).
- Guías de despliegue con Docker y Render.
