Welcome to the official **Fitness App** documentation.

This project is a full-stack fitness and health tracking app that includes:

- A **Node.js/Express backend** with **Drizzle ORM** and **PostgreSQL**.
- A modern **React (Vite) frontend**.
- A rich **relational database schema** for users, training, nutrition, goals, notifications, achievements and chat.

In this site you will find:

- How to install and run the project (locally and in production).
- Backend and frontend architecture.
- Full database schema and relations.
- Detailed API reference (endpoints, payloads, responses).
- Docker and Render deployment guides.

If you are new to the project, start with:

- [`getting-started/project-overview`](getting-started/project-overview.md)
